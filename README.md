# Banking System Simulation 🏦

An interactive, web-based banking simulator application designed using core Java logic and the Model-View-Controller (MVC) architecture under Spring Boot. This project automates essential financial operations, emphasizing clean object-oriented program programming (OOP) principles and robust application data flow.

## 🛠️ Technological Framework

The application is engineered using a decoupled, layered architecture to maintain clear modularity:

- **Backend Logic:** Java 17, Spring Boot 3.x
- **Build & Dependency Management:** Apache Maven
- **Frontend UI Layer:** HTML5, CSS3, Thymeleaf Template Engine
- **Embedded Web Server:** Apache Tomcat
